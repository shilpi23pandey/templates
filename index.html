Synergy: Disaster Relief

A webApp that coordinates and manages the functioning of disaster relief agencies and will also help them identify the regions which require immediate mitigation

How this works?

A webApp that coordinates and manages the functioning of disaster relief agencies and will also help them identify the regions which require immediate mitigation

  • Problem : There is no proper coordination and collaboration between agencies working for disaster relief.
  • Crowdsourced data (tweets,user data from the app,media data etc.,) is used to classify disaster hit regions into high impact, medium impact etc., also user requests from the application can be used to specify the required resource (e.g., water, food, shelter etc.,) in the disaster hit zone. Inferences from crowdsourced data is validated by other users of the applications using proof of work concept and a impact map will be generated. This impact map may be used by agencies to provide relief. Multiple agencies should not respond to an individual request at the same time so app will partition the user requests and will assign the request to a particular agency based on area and resource needed
  • In the event of a crisis, a user can send a request for help using the app. They can also specify the resources required by them. If the users are not able to send requests themselves due to connectivity issues, someone from a remote location, their family or friends can send the request. Requests are validated and accessed to generate a impact map for the agencies and concerned agencies may start relief.
